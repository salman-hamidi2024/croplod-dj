شرایط و قوانین استفاده از وب‌سایت کراپلود بات، تلگرام و ربات‌های تلگرامی

این سند شامل تمامی شرایط و قوانین مربوط به استفاده از خدمات وب‌سایت کراپلود بات، پلتفرم تلگرام، و ربات‌های ساخته‌شده از طریق این وب‌سایت یا تلگرام است. این قوانین به‌منظور ایجاد محیطی امن، قانونی و کاربرپسند برای تمامی کاربران طراحی شده‌اند. لطفاً پیش از استفاده از خدمات، این شرایط را به‌دقت مطالعه کنید. استفاده از خدمات به‌منزله پذیرش کامل این قوانین و هرگونه به‌روزرسانی آینده آن است. در صورت عدم پذیرش این شرایط، لطفاً از استفاده از خدمات خودداری کنید.

قوانین کلی وب‌سایت کراپلود بات

وب‌سایت کراپلود بات با هدف ارائه خدمات ساخت، مدیریت و بهینه‌سازی ربات‌های تلگرامی طراحی شده است. این پلتفرم امکان ایجاد ربات‌هایی با قابلیت‌های متنوع مانند اطلاع‌رسانی، مدیریت گروه‌ها، ارائه خدمات تجاری و غیره را فراهم می‌کند. کاربران موظف به رعایت تمامی قوانین زیر هستند تا از عملکرد صحیح و امن خدمات اطمینان حاصل شود:

  • کاربران باید قوانین جمهوری اسلامی ایران و مقررات بین‌المللی مرتبط با استفاده از خدمات آنلاین و تلگرام را رعایت کنند. این شامل رعایت قوانین مربوط به حریم خصوصی، مالکیت فکری و محتوای دیجیتال است.

  • هرگونه فعالیت غیرقانونی، از جمله نقض حریم خصوصی، انتشار محتوای غیرمجاز (مانند محتوای مستهجن، خشونت‌آمیز، یا توهین‌آمیز)، یا انجام اقدامات مخرب مانند هک یا انتشار بدافزار، ممنوع است و ممکن است به پیگیری قانونی منجر شود.

  • کاربران مسئول حفظ امنیت حساب کاربری، رمز عبور و اطلاعات ورود خود هستند. کراپلود هیچ مسئولیتی در قبال سوءاستفاده از حساب‌های کاربری به دلیل سهل‌انگاری کاربران (مانند استفاده از رمز عبور ضعیف یا به‌اشتراک‌گذاری اطلاعات ورود) ندارد.

  • تلاش برای هک، نفوذ، یا ایجاد اختلال در عملکرد وب‌سایت، سرورها، یا خدمات، از جمله ارسال درخواست‌های غیرمجاز یا حملات سایبری، ممنوع است و منجر به مسدود شدن حساب کاربری و پیگیری قانونی خواهد شد.

  • کاربران باید اطلاعات صحیح و معتبر (مانند نام، ایمیل، و شماره تماس) در هنگام ثبت‌نام ارائه دهند. ارائه اطلاعات نادرست ممکن است به تعلیق حساب یا محدودیت دسترسی به خدمات منجر شود.

  • کاربران نباید از خدمات وب‌سایت برای اهداف غیراخلاقی، مانند کلاهبرداری، فیشینگ، یا انتشار بدافزار، استفاده کنند. این شامل استفاده از ربات‌ها برای جمع‌آوری غیرمجاز اطلاعات کاربران یا ارسال پیام‌های ناخواسته است.

  • کاربران باید از انتشار محتوایی که به گروه‌های خاص توهین می‌کند، مانند محتوای نژادپرستانه، مذهبی توهین‌آمیز، یا تبعیض‌آمیز، خودداری کنند. کراپلود حق حذف چنین محتوایی را بدون اطلاع قبلی دارد.

  • استفاده از خدمات کراپلود برای فعالیت‌های تجاری غیرقانونی، مانند فروش محصولات غیرمجاز یا نقض قوانین تجارت الکترونیک، ممنوع است.

  • کاربران باید از ایجاد حساب‌های متعدد برای دور زدن محدودیت‌های پلن‌های رایگان یا پولی خودداری کنند. این اقدام ممکن است به مسدود شدن تمامی حساب‌های مرتبط منجر شود.

  • هرگونه تلاش برای مهندسی معکوس، کپی‌برداری از کد منبع وب‌سایت، یا سوءاستفاده از زیرساخت‌های کراپلود ممنوع است و عواقب قانونی خواهد داشت.

قوانین پلتفرم تلگرام

تلگرام به‌عنوان یک پیام‌رسان جهانی با میلیون‌ها کاربر، شرایط استفاده‌ای دارد که برای کاربران، توسعه‌دهندگان و مدیران ربات‌ها الزام‌آور است. این قوانین در وب‌سایت رسمی تلگرام و بخش‌های مرتبط با حریم خصوصی و API ربات‌ها شرح داده شده‌اند. کاربران کراپلود بات که از خدمات تلگرام استفاده می‌کنند، ملزم به رعایت این قوانین هستند:

  • کاربران نباید محتوای غیرقانونی، مانند محتوای خشونت‌آمیز، مستهجن، یا نقض‌کننده حقوق مالکیت فکری (مانند انتشار بدون اجازه آثار دارای کپی‌رایت)، منتشر کنند. این شامل تصاویر، ویدئوها، یا متون توهین‌آمیز است.

  • تلگرام از پروتکل رمزنگاری MTProto برای ارتباطات امن استفاده می‌کند. پیام‌های قدیمی پس از پردازش از سرورهای ابری تلگرام حذف می‌شوند تا حریم خصوصی کاربران حفظ شود. کاربران باید از تنظیمات حریم خصوصی تلگرام برای کنترل دسترسی به اطلاعات خود استفاده کنند.

  • ارسال پیام‌های انبوه تبلیغاتی (اسپم)، انتشار محتوای جعلی یا گمراه‌کننده، یا فعالیت‌های مخرب مانند ارسال لینک‌های فیشینگ، ممنوع است و ممکن است به مسدود شدن حساب کاربری منجر شود.

  • ربات‌ها در حالت Privacy Mode فقط به پیام‌هایی که مستقیماً برایشان ارسال شده یا با "/" شروع می‌شوند دسترسی دارند، مگر اینکه این حالت توسط مدیر گروه غیرفعال شود. این تنظیم برای حفاظت از حریم خصوصی اعضای گروه‌ها طراحی شده است.

  • تلگرام حق حذف محتوا، مسدود کردن حساب‌ها، یا غیرفعال کردن ربات‌های متخلف را بدون اطلاع قبلی دارد. این شامل ربات‌هایی است که قوانین تلگرام یا قوانین محلی کشورها را نقض می‌کنند.

  • کاربران باید از ابزارهای ضداسپم تلگرام، مانند تنظیمات گروه یا فیلترهای پیام، برای جلوگیری از فعالیت ربات‌های مزاحم یا پیام‌های ناخواسته استفاده کنند. برای مثال، می‌توانند ربات‌های ناشناخته را محدود کرده یا گزارش دهند.

  • تلگرام از کاربران انتظار دارد که از ابزارهای امنیتی مانند احراز هویت دو مرحله‌ای (Two-Step Verification) برای حفاظت از حساب خود استفاده کنند. این ویژگی از طریق تنظیمات تلگرام قابل فعال‌سازی است.

  • کاربران نباید از تلگرام برای فعالیت‌هایی که به نقض قوانین محلی یا بین‌المللی منجر می‌شود، مانند سازمان‌دهی فعالیت‌های غیرقانونی یا انتشار اطلاعات حساس، استفاده کنند.

  • تلگرام هیچ‌گونه دسترسی به چت‌های رمزنگاری‌شده Secret Chat ندارد و این چت‌ها فقط روی دستگاه کاربران ذخیره می‌شوند، نه در سرورهای ابری تلگرام.

  • کاربران باید از انتشار اطلاعات نادرست یا گمراه‌کننده که می‌تواند به ضرر دیگران منجر شود، مانند اخبار جعلی یا محتوای غیرعلمی در حوزه سلامت، خودداری کنند.

قوانین ساخت ربات در کراپلود بات

کراپلود بات بستری برای ساخت و مدیریت ربات‌های تلگرامی ارائه می‌دهد که امکان ایجاد ربات‌هایی با قابلیت‌های متنوع را فراهم می‌کند. کاربران باید شرایط زیر را برای ایجاد، مدیریت و استفاده از ربات‌ها رعایت کنند تا از عملکرد صحیح و قانونی ربات‌ها اطمینان حاصل شود:

  • ربات‌ها نباید برای انتشار محتوای غیرقانونی، توهین‌آمیز، یا نقض‌کننده حقوق دیگران (مانند کپی‌رایت، حریم خصوصی، یا علامت تجاری) استفاده شوند. برای مثال، ربات‌هایی که محتوای دارای کپی‌رایت را بدون اجازه توزیع می‌کنند، غیرمجاز هستند.

  • توکن ربات باید از طریق BotFather در تلگرام دریافت شده و معتبر باشد. کاربران نباید از توکن‌های غیرمعتبر یا سرقت‌شده استفاده کنند.

  • کاربران باید محدودیت‌های پلن انتخابی خود (مانند تعداد دکمه‌ها، حجم آپلود فایل، تعداد درخواست‌های روزانه، یا قابلیت‌های پیشرفته) را رعایت کنند. تخطی از این محدودیت‌ها ممکن است به تعلیق خدمات منجر شود.

  • استفاده از ربات برای ارسال اسپم، تبلیغات ناخواسته، یا فعالیت‌های مخرب (مانند حملات DDoS یا جمع‌آوری غیرمجاز داده‌ها) ممنوع است و منجر به غیرفعال شدن ربات و حساب کاربری خواهد شد.

  • کراپلود حق دارد ربات‌هایی که قوانین وب‌سایت یا تلگرام را نقض کنند، بدون اطلاع قبلی غیرفعال یا حذف کند. این شامل ربات‌هایی است که محتوای غیرقانونی منتشر می‌کنند یا باعث اختلال در خدمات می‌شوند.

  • کاربران باید از ایجاد ربات‌هایی که بار سرور را بیش‌ازحد افزایش می‌دهند (مانند ربات‌های با درخواست‌های مکرر و غیرضروری یا اجرای دستورات پیچیده بدون بهینه‌سازی) خودداری کنند.

  • هر ربات باید دارای توضیحات واضح و کاربردی در بخش معرفی ربات (About) باشد تا کاربران نهایی دچار سردرگمی نشوند. این توضیحات باید هدف و قابلیت‌های ربات را به‌صورت شفاف بیان کنند.

  • کاربران باید از ایجاد ربات‌هایی که قابلیت‌های غیرمجاز (مانند دانلود غیرقانونی محتوا، دسترسی به اطلاعات خصوصی بدون رضایت، یا نقض قوانین محلی) ارائه می‌دهند، خودداری کنند.

  • ربات‌ها باید به‌گونه‌ای طراحی شوند که تجربه کاربری مثبتی ارائه دهند، مانند پاسخ‌گویی سریع، دستورات ساده و قابل‌فهم، و رابط کاربری مناسب.

  • کاربران باید از ایجاد ربات‌هایی که به‌صورت خودکار کاربران را به گروه‌ها یا کانال‌های دیگر اضافه می‌کنند، بدون رضایت صریح آن‌ها، خودداری کنند.

قوانین ساخت ربات‌های تلگرامی (بر اساس API تلگرام)

توسعه‌دهندگان ربات‌های تلگرامی باید از Bot API تلگرام و استانداردهای آن پیروی کنند. این API امکاناتی مانند ارسال پیام، مدیریت گروه‌ها، پرداخت آنلاین، و بازی‌های تعاملی را فراهم می‌کند. جزئیات این قوانین عبارت‌اند از:

  • ربات‌ها باید از طریق BotFather ایجاد شوند و از توکن معتبر برای ارتباط با سرورهای تلگرام استفاده کنند. توکن‌ها باید محرمانه نگه داشته شوند و در دسترس اشخاص غیرمجاز قرار نگیرند.

  • توسعه‌دهندگان باید از ارسال درخواست‌های بیش‌ازحد به API تلگرام (Rate Limiting) خودداری کنند. برای مثال، محدودیت‌های API شامل حداکثر 30 درخواست در ثانیه برای یک ربات یا 20 پیام در دقیقه به یک کاربر است.

  • ربات‌ها نباید اطلاعات خصوصی کاربران (مانند شماره تلفن، آدرس ایمیل، یا پیام‌های خصوصی) را بدون رضایت صریح جمع‌آوری یا ذخیره کنند. این شامل رعایت قوانین GDPR در اروپا است.

  • ربات‌ها باید با پروتکل HTTPS و Bot API تلگرام کار کنند و از پروتکل رمزنگاری‌شده MTProto پشتیبانی کنند تا امنیت ارتباطات حفظ شود.

  • تلگرام می‌تواند ربات‌هایی که قوانین را نقض کنند، غیرفعال یا حذف کند. این شامل ربات‌هایی است که محتوای غیرقانونی منتشر می‌کنند یا باعث اختلال در عملکرد سرورهای تلگرام می‌شوند.

  • ربات‌ها باید قابلیت‌های مشخص و کاربردی (مانند اطلاع‌رسانی، مدیریت گروه، خدمات تجاری، یا بازی‌های تعاملی) ارائه دهند و از ایجاد ربات‌های بی‌هدف یا غیرکاربردی خودداری شود.

  • توسعه‌دهندگان باید از Webhook یا Polling برای دریافت به‌روزرسانی‌های ربات استفاده کنند و سرورهای پایدار با گواهینامه SSL معتبر برای میزبانی انتخاب کنند.

  • ربات‌ها باید به‌گونه‌ای طراحی شوند که با نسخه‌های جدید API تلگرام سازگار باشند. توسعه‌دهندگان باید مستندات تلگرام را به‌طور منظم بررسی کنند تا از تغییرات API مطلع شوند.

  • کاربران باید از ایجاد ربات‌هایی که به‌صورت خودکار کاربران را به گروه‌ها یا کانال‌های دیگر دعوت می‌کنند، بدون رضایت آن‌ها، خودداری کنند.

  • توسعه‌دهندگان باید از ابزارهای تحلیلی تلگرام (مانند Bot Analytics) برای بررسی عملکرد ربات و بهبود تجربه کاربری استفاده کنند.

حریم خصوصی و امنیت

کراپلود بات و تلگرام متعهد به حفاظت از حریم خصوصی و امنیت اطلاعات کاربران هستند. سیاست‌های حریم خصوصی و اقدامات امنیتی ما به‌گونه‌ای طراحی شده‌اند که اعتماد کاربران را حفظ کنند و مطابق با استانداردهای جهانی مانند GDPR عمل کنند. جزئیات این سیاست‌ها به شرح زیر است:

  • اطلاعات شخصی کاربران (مانند نام، ایمیل، شماره تماس، یا اطلاعات پرداخت) نزد کراپلود محفوظ است و به اشخاص ثالث فروخته یا منتقل نمی‌شود، مگر با رضایت صریح کاربر یا الزام قانونی مانند احکام قضایی.

  • داده‌های ربات‌ها (مانند توکن، تنظیمات، یا پیام‌های پردازش‌شده) به‌صورت رمزنگاری‌شده با استانداردهای AES-256 ذخیره می‌شوند و فقط توسط کاربران مجاز قابل دسترسی هستند.

  • کاربران می‌توانند در هر زمان از طریق صفحه تماس با ما درخواست حذف کامل اطلاعات حساب کاربری خود را ارائه دهند. این درخواست ظرف مدت 30 روز پردازش خواهد شد.

  • کراپلود از کوکی‌ها و فناوری‌های مشابه برای بهبود تجربه کاربری (مانند ذخیره تنظیمات، تحلیل رفتار کاربر، یا شخصی‌سازی خدمات) استفاده می‌کند. کاربران می‌توانند کوکی‌ها را از طریق تنظیمات مرورگر غیرفعال کنند، اما این ممکن است برخی قابلیت‌ها را محدود کند.

  • تلگرام پیام‌های قدیمی را پس از پردازش از سرورهای ابری حذف می‌کند. ربات‌ها در حالت Privacy Mode فقط به پیام‌های مستقیم یا پیام‌هایی که با "/" شروع می‌شوند دسترسی دارند، مگر اینکه مدیر گروه این حالت را غیرفعال کند.

  • کراپلود از فایروال‌های پیشرفته، پروتکل HTTPS، و سیستم‌های تشخیص نفوذ (IDS) برای حفاظت از سرورها و داده‌های کاربران استفاده می‌کند. این سیستم‌ها به‌طور مداوم به‌روزرسانی می‌شوند.

  • کاربران باید از رمزهای عبور قوی (حداقل 12 کاراکتر، شامل حروف بزرگ و کوچک، اعداد، و نمادها) استفاده کنند و از به‌اشتراک‌گذاری توکن ربات یا اطلاعات ورود خودداری کنند.

  • کراپلود هیچ‌گونه دسترسی غیرمجاز به پیام‌های کاربران در تلگرام ندارد و فقط داده‌های لازم برای ارائه خدمات (مانند تنظیمات ربات یا تاریخچه فعالیت) را پردازش می‌کند.

  • در صورت وقوع نقض امنیتی، کراپلود فوراً کاربران را از طریق ایمیل یا اعلان وب‌سایت مطلع کرده و اقدامات لازم مانند بازنشانی رمز عبور یا مسدود کردن دسترسی‌های غیرمجاز را انجام خواهد داد.

  • کاربران باید از ربات‌هایی که درخواست دسترسی غیرضروری به اطلاعات (مانند شماره تلفن، موقعیت مکانی، یا داده‌های حساس) دارند، اجتناب کنند و آن‌ها را به تلگرام گزارش دهند.

  • تلگرام از کاربران انتظار دارد که تنظیمات حریم خصوصی خود را (مانند مخفی کردن شماره تلفن یا محدود کردن دسترسی به پروفایل) بررسی و به‌روزرسانی کنند تا از سوءاستفاده جلوگیری شود.

  • کراپلود ممکن است برای تحلیل رفتار کاربران و بهبود خدمات، داده‌های ناشناس (مانند تعداد درخواست‌های ربات یا الگوهای استفاده) را جمع‌آوری کند، اما این داده‌ها به‌هیچ‌عنوان به هویت کاربران مرتبط نمی‌شوند.

پشتیبانی و مسئولیت‌ها

کراپلود بات و تلگرام متعهد به ارائه خدمات باکیفیت و پشتیبانی مناسب هستند، اما مسئولیت استفاده از خدمات و ربات‌ها بر عهده کاربران است. این بخش جزئیات بیشتری درباره پشتیبانی، مسئولیت‌ها و تعهدات ارائه می‌دهد:

  • پشتیبانی کراپلود از طریق ایمیل، شماره تماس، یا چت آنلاین در دسترس است. در پلن‌های پیشرفته‌تر، پشتیبانی 24/7 و اولویت‌دار ارائه می‌شود.

  • کراپلود و تلگرام هیچ مسئولیتی در قبال قطعی‌های احتمالی ناشی از مشکلات سرورهای تلگرام، ارائه‌دهندگان هاست، یا سایر سرویس‌های خارجی (مانند قطعی اینترنت یا مشکلات DNS) ندارند.

  • در صورت تغییر قوانین یا شرایط خدمات، کاربران از طریق ایمیل یا اعلان در وب‌سایت مطلع خواهند شد. ادامه استفاده از خدمات به‌منزله پذیرش تغییرات است.

  • کراپلود و تلگرام هیچ مسئولیتی در قبال خسارات ناشی از استفاده نادرست از ربات‌ها، مانند انتشار محتوای غیرقانونی، نقض حریم خصوصی، یا خسارات مالی ناشی از سوءاستفاده کاربران، ندارند.

  • کاربران باید از ربات‌های ضداسپم یا تنظیمات گروه در تلگرام برای جلوگیری از ورود ربات‌های مزاحم یا پیام‌های ناخواسته استفاده کنند. برای مثال، می‌توانند دسترسی ربات‌ها به گروه را محدود کنند.

  • کاربران مسئول بررسی و به‌روزرسانی ربات‌های خود برای سازگاری با نسخه‌های جدید API تلگرام هستند. کراپلود ابزارها و راهنمایی‌هایی برای به‌روزرسانی ربات‌ها ارائه می‌دهد.

  • کراپلود ممکن است در صورت نیاز به‌روزرسانی‌هایی در خدمات، ویژگی‌ها، یا قوانین اعمال کند. کاربران موظف به پذیرش این تغییرات یا توقف استفاده از خدمات هستند.

  • کاربران باید از ربات‌های خود به‌گونه‌ای استفاده کنند که باعث اختلال در تجربه سایر کاربران (مانند ارسال پیام‌های بیش‌ازحد یا ایجاد محتوای نامناسب) نشود.

  • در صورت بروز مشکلات فنی، کاربران باید ابتدا مستندات کراپلود و تلگرام را بررسی کنند و سپس از طریق پشتیبانی درخواست کمک کنند.

نکات تکمیلی و توصیه‌ها

برای استفاده بهینه از خدمات کراپلود بات و تلگرام و جلوگیری از مشکلات احتمالی، رعایت نکات زیر توصیه می‌شود:

  • برای اطلاعات بیشتر درباره ساخت ربات، به مستندات تلگرام مراجعه کنید. این مستندات شامل راهنمایی‌های جامع برای توسعه ربات‌ها و استفاده از API است.

  • کاربران باید از به‌روزرسانی‌های تلگرام و کراپلود بات مطلع باشند و تنظیمات ربات‌های خود را مطابق با تغییرات جدید هماهنگ کنند. تلگرام مرتباً قابلیت‌های جدیدی به API خود اضافه می‌کند.

  • کاربران باید از ابزارهای تست و دیباگ (مانند BotFather یا ابزارهای توسعه‌دهنده تلگرام) برای بررسی عملکرد ربات‌های خود استفاده کنند تا از خطاها جلوگیری شود.

  • توصیه می‌شود کاربران از سرورهای میزبانی با پایداری بالا و پهنای باند مناسب برای ربات‌های خود استفاده کنند تا از قطعی خدمات جلوگیری شود.

  • کاربران باید از ایجاد ربات‌هایی که باعث سردرگمی کاربران می‌شوند (مانند ربات‌هایی با دستورات پیچیده یا بدون راهنما) خودداری کنند و تجربه کاربری ساده و شفافی ارائه دهند.
